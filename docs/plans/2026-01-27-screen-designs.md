## Review Decisions Log

**Date:** 2026-01-27

Decisions made during screen-by-screen review:

| Screen | Decision |
|--------|----------|
| Dashboard | Minimal stepper (dots only, no labels), sort by latest updated, filter by stage status |
| Quick Convert Landing | 2x2 grid layout, no badges, mockups included |
| QC - Requirements | Inline editing enabled, button priority: Save > Generate PRD > Download, file types: .txt and .md only |
| QC - PRD | Support both JSON and bullet-point input, section streaming, simple textarea for edit |
| QC - User Stories | All 3 formats (Standard, Gherkin, JTBD), drag-and-drop reorder, include Priority option |
| QC - Mockups | Defer sketch upload, allow multi-device selection, generate 3 variations |
| Project Shell | Use breadcrumbs (not just back link), settings: rename/delete/archive, show stage labels on mobile |
| Project - Requirements | Add Meeting via modal, single item add, fixed section order |
| Project - PRD | Warn on regenerate, show version history, PRD type selectable |
| Project - User Stories | Generate More appends (not replaces), user-defined labels, show priority count in summary |
| Project - Mockups | Link mockups to stories, allow renaming, let users select which stories to visualize |
| Project - Export | v1: download only (no Jira), markdown as zip file, export history deletable |
